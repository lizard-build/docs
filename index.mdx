# Introduction

Welcome to the **Lizard** documentation.

## What is Lizard?

Lizard is a developer platform for building, deploying, and scaling modern applications. It provides a unified CLI, infrastructure orchestration, and real-time observability — so your team ships faster with fewer moving parts.

## Why Lizard?

- **Zero-config deployments** — push code, Lizard handles the rest
- **Edge-first** — your app runs close to your users by default
- **Built-in observability** — logs, metrics, and traces out of the box
- **Framework agnostic** — works with Next.js, Remix, Astro, or any Node.js app
